# BraInha Server

> 교내 학술 정보 교류 플랫폼

## Installation

- Clone the repo:

```bash
git clone https://github.com/Astro36/Kokoa.git
```

## Build Setup

```bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run start
```
